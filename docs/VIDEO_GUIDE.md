# 安安虾演示视频 - 快速录制指南

## 🎯 目标
制作 60-90 秒的高质量演示视频，展示安安虾核心功能

## 📱 录制方式选择

### 方式 1: 终端直接录制 (推荐)
```bash
cd /home/kyne/.openclaw/workspace/anansha
./scripts/record_demo.sh
```
同时用录屏软件录制终端画面

### 方式 2: 使用 asciinema + 后期编辑
```bash
# 录制终端操作
asciinema rec demo.cast

# 转换为视频
asciicast2gif demo.cast demo.gif
# 或使用 asciinema-player 导出
```

### 方式 3: 分镜录制
手动执行每个命令，录屏后剪辑拼接

## 🎨 视觉规范

### 颜色
- 背景: `#1A1A1A` (深色)
- 主色: `#F0B90B` (币安黄)
- 强调: `#00D084` (成功绿)
- 警示: `#FF6B6B` (警告红)
- 文字: `#FFFFFF` (白色)

### 字体
- 标题: 28-32px Bold
- 正文: 14-16px Regular
- 代码: JetBrains Mono 13px

### 动画
- 过渡: 0.3s ease
- 打字效果: 30ms/字符
- 避免过度动画
