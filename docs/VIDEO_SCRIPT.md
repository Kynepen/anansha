## 🎨 视觉风格

- **动画:** 简洁流畅，避免过度花哨

## 🎥 录制指南

### 终端录制
```bash
# 方式1: asciinema (推荐)
asciinema rec demo.cast

# 方式2: 直接录屏
# 使用 OBS / QuickTime / 其他录屏软件
```

### 录屏设置
- **分辨率:** 1920x1080
- **帧率:** 30fps
- **码率:** 5000kbps
- **终端字体:** 14-16pt，确保清晰可读

### 编辑软件推荐
- **免费:** DaVinci Resolve, CapCut
- **付费:** Adobe Premiere Pro, Final Cut Pro
